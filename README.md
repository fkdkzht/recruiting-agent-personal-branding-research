## このプロジェクトについて

採用エージェント事業における**事業部代表の個人ブランディング戦略**について、アイディア出し・リサーチ・分析・インサイト抽出を行い、**事業部代表に提出するレポート**としてアウトプットするための作業スペースです。

## 進め方（運用ルール）

- `00_inbox/`: 雑に放り込む場所（URL、メモ、スクショ、思いつきなど）
- `01_research/`:
  - `sources/`: ソース（記事、論文、書籍メモ、競合の公開情報など）
  - `notes/`: 読み解きメモ（要約、引用、仮説、反証）
- `02_ideation/`: 戦略案の発散（施策案、仮説、論点、打ち手の組み合わせ）
- `03_analysis/`: フレームでの整理（ターゲット、ポジション、メッセージ、チャネルなど）
- `04_insights/`: インサイト（示唆、意思決定ポイント、優先順位、リスク）
- `05_outputs/`: 提出物（レポート、スライド、要点サマリ）
- `99_archive/`: ボツ案、古い版、採用しなかった素材

## まず作る成果物

- `05_outputs/report_outline.md`: レポート構成（叩き台）
