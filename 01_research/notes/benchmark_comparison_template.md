## 列定義（全行共通）

| 列ID | 意味 | メモ |
| --- | --- | --- |
| subject | 対象名（個人） | 表記ゆれを最初の1行で統一 |
| service | 事業・サービス名 | 法人名／ブランド名 |
| promise | 候補者（または顧客）への**約束** | 何を届けると言っているか |
| trust_basis | **信頼の根拠** | 実績・専門性・透明性・人格・第三者評価など、主軸を明示 |
| funnel | **導線** | 認知チャネル → 低摩擦入口 → 商談／申込まで |
| risk | **リスク** | 拒絶・炎上・期待差・コンプラ・利益相反の見せ方 |
| scale | **スケール** | 個人→組織への移管、品質の守り方 |
| evidence | **一次情報の根拠** | URL 箇条書き（`sources/` のファイル名でも可） |
| notes | 補足 | 推測は **（推測）** と明記 |

### 田代正義 / エンジニアのミカタ

- **subject**: 田代正義（株式会社エンジニアのミカタ 代表取締役／note プロフィール表記）
- **service**: 株式会社エンジニアのミカタ（コーポレートサイト表記）
- **promise**  
  - コーポレートのミッション見出しどおり、「エンジニアの生涯を 1億円UPし、真にエンジニアのミカタとなる」ことを掲げる。  
  - 日本のSES業界の構造問題（多重下請け、成果に見合わない年収、エンジニアを商品視する営業）を問題提起し、エンジニアが報われる社会・憧れの職業へ、と宣言。  
  - 「選ばれる理由」として、前職比平均170万円以上UP、高還元率83%と単価連動型評価、案件選択制度（エンジニアが100%選択）、参画後のフルサポート（ヒアリング・単価交渉・キャリア相談）、ワークライフバランスを列挙。  
  - note プロフィールでは「エンジニアの生涯所得最大化をしています。」（一次情報）
- **trust_basis**  
  - 数値と制度（還元率、案件数、事例スキル・金額）での証拠づくり。  
  - 取引実績・株主（複数ファンド・個人名）・メディア掲載の提示。  
  - 代表個人は note・X でフォロワー獲得と思想の入口になりうる構成（中間レポートの「思想→事業」と整合）。
- **funnel**  
  - 認知: note、X、法人サイト、メディア記事（サイト内リンク一覧）。  
  - 低摩擦: 「LINEでカジュアル面談」「フォームでカジュアル面談」。「市場価値の確認だけでも歓迎」。  
  - 深いコミット: 「エントリーする」で選考希望者向け。  
  - 出典: `https://engineer-mikata.com/` トップ各セクション。
- **risk**  
  - 金額・年収UPの強調は、候補者側で「エージェントの煽り」と解釈されるリスク。（推測）  
  - 業界批判トーンは支持と拒絶の両極。（推測）  
  - 事例は匿名イニシャル表記。再現性・個人差の説明が薄いと期待差リスク。（推測）
- **scale**  
  - コーポレートは組織・制度・資本（株主）で信頼のスケールを補強。代表は思想・物語の前面担当。（推測）  
  - 従業員規模・組織図は別ページの一次確認推奨（未確認）。

### IT菩薩モロー（毛呂淳一朗） / キッカケエージェント

- **subject**: IT菩薩モロー（毛呂淳一朗）
- **service**: キッカケエージェント
- **promise**  
  - エンジニア特化の転職エージェントとして、キャリア相談・キャリアパス助言を行う。  
  - 企業が利用している技術スタックや環境を踏まえた「驚異的なマッチング」を訴求。  
  - 応募企業に合わせた面接対策（質問/評価観点レベル）を行い、高い内定率をうたう。  
  - 出典: `https://kikkakeagent.com/lp/` の各見出し（キャリアプラン／マッチング／レジュメ・面接対策）。
- **trust_basis**  
  - 「IT人事YouTuberのモローがプロデュース」「エンジニア採用担当としての経験」などの経歴提示。  
  - 内定率（一般的エージェントの2倍以上）という数値訴求、および転職成功事例の掲載。  
  - 出典: `https://kikkakeagent.com/lp/`（「キッカケエージェントの転職支援は…」「IT菩薩モローとは」「転職成功事例」）。
- **funnel**  
  - 認知: LP（オウンド）での価値訴求／（推測）YouTube等の発信。  
  - 低摩擦: 「無料転職サポート登録」または「LINE登録（面接対策動画配布）」で接点を作る。  
  - 出典: `https://kikkakeagent.com/lp/` のCTA（無料登録・LINE登録）。
- **risk**  
  - 「最強」「驚異的」など強い言い切りは期待値が上がりすぎるリスク。（推測）  
  - 個人（YouTuber）起点の場合、属人性や炎上耐性が事業に直撃しやすい。（推測）
- **scale**  
  - LP 上はプロセス（キャリアプラン→マッチング→面接対策）として提示されており、一定の型化余地はある。  
  - ただし品質が“誰が担当しても同品質か”は一次情報では未確認。（推測）
